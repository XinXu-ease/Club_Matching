# 🚀 三种身份用户流程快速查询锦囊

## 一句话总结

| 身份 | 定位 | 导航栏 | 核心能力 |
|------|------|--------|----------|
| **👤 游客** | 浏览者 | 无下栏 | 只读 Explore |
| **🎓 新生** | 参与者 | E \| M \| Ms \| P | 报名+匹配+提问 |
| **🏛️ 管理员** | 运营者 | E \| Mg \| Ms \| P | 发布+审核+统计 |

## 🌳 两个完整的用户故事

### 故事 1：张三是新生，想找个适合自己的社团

```
1️⃣ 首次进入 ClubMatch
   └─ 欢迎页 (s-welcome)
   
2️⃣ 点击 "我是新生"
   └─ 跳转到登录/注册 (s-login / s-signup)
   
3️⃣ 登录成功后，进入 Explore 首页 (s-explore)
   ├─ 底部导航栏出现：Explore | Match | Messages | Profile
   ├─ 看到热门社团卡片、近期活动、正在招新
   └─ 还看到一个大 banner：高亮提示 "还不知道加入哪个社团？去做匹配测试"

4️⃣ 张三有两条路：

   路径 A - 直接浏览和报名：
   ├─ 点击某个社团卡片 → s-club (社团主页)
   ├─ 点击 ❤ 【收藏】→ 保存到 Profile
   ├─ 点击 📝【报名】→ s-apply (报名表)
   └─ 填表并提交 → s-apply-success (成功页)

   路径 B - 先做匹配再报名（推荐新生用）：
   ├─ 点击 "去做匹配" banner → s-match-guide (匹配引导)
   ├─ 点击 【开始匹配】
   ├─ 进入 s-match-quiz (4步问答)
   │  ├─ 第1步：选择感兴趣的领域 (艺术/学术/公益/技术...)
   │  ├─ 第2步：选择参与目标 (拓展人脉/积累经历/打比赛...)
   │  ├─ 第3步：选择时间投入 (高/中/低)
   │  └─ 第4步：选择个人风格 (大组织/小圈子/独立/合作...)
   ├─ 点击 【提交】
   └─ 查看 s-match-result (推荐结果)
      ├─ 显示你的用户画像："低投入 + 表达型 + 小圈子友好"
      ├─ 推荐 3-5 个最匹配的社团
      ├─ 每个社团显示匹配度和原因
      └─ 可点击【查看主页】、【收藏】、【报名】

5️⃣ 张三的 Profile (s-profile)
   ├─ 显示头像、昵称、个人标签
   ├─ Tab 1 - 我的收藏：显示 ❤ 收藏过的所有社团
   ├─ Tab 2 - 我的报名：显示 📝 已报名的社团和审核状态
   ├─ Tab 3 - 浏览记录：显示 👀 最近浏览过的社团/活动
   ├─ Tab 4 - 我的提问：显示 💬 问过的问题和社团的回复
   └─ Settings 按钮：可以看到 【申请社团管理员身份】选项

6️⃣ 如果张三想升级成管理员（假设他要创办一个新社团）
   ├─ 点击 Profile → Settings → 【申请社团管理员】
   ├─ 进入 s-auth-apply (认证申请表)
   │  ├─ 填写社团名称
   │  ├─ 填写职位
   │  ├─ 上传身份证件
   │  ├─ 上传社团证书/任命书
   │  ├─ 填写电话
   │  └─ 点击 【提交申请】
   ├─ 进入 s-auth-status (审核进度页)
   │  ├─ 显示 ✅ 申请已提交
   │  ├─ 显示 ⏳ 审核中（3-7个工作日）
   │  └─ 显示 ⏳ 审核完成（待通知）
   └─ 🔔 审核通过后
      ├─ 收到通知
      ├─ 回到 Profile → Settings
      ├─ 现在能看到 【切换身份】选项
      └─ 点击切换 → 身份变为"社团管理员"

7️⃣ 切换成管理员后的变化
   ├─ 底部导航栏变为：Explore | **Manage** (替代 Match) | Messages | Profile
   ├─ Profile 页面变为 s-profile-admin (管理员版本)
   │  ├─ 显示当前管理的社团名：🏛️ 校园戏剧社
   │  ├─ 显示认证状态：✅ 已认证
   │  └─ 显示运营数据速览
   └─ 可以进入 s-manage (管理台首页)
      ├─ 看到 4 个快捷卡片：编辑主页、发布招新、查看报名、管理留言
      ├─ 看到数据卡片：访问量、收藏量、报名数、待回复留言
      └─ 可以开始运营社团了！
```

### 故事 2：李四是社团负责人，想通过平台运营和招新

```
1️⃣ 首次进入 ClubMatch
   └─ 欢迎页 (s-welcome)
   
2️⃣ 点击 "我是社团负责人"
   └─ 跳转到登录/注册

3️⃣ 登录成功后，自动跳转到 s-auth-apply (认证申请表)
   ├─ 系统提示："请先完成身份认证，即可获得完整的社团管理权限"
   ├─ 填写社团信息
   │  ├─ 社团名称：校园戏剧社
   │  ├─ 职位：主席
   │  ├─ 上传证书
   │  ├─ 联系电话
   │  └─ 社团简介（可选）
   └─ 点击 【提交申请】

4️⃣ 进入认证等待阶段 (s-auth-status)
   ├─ 显示申请信息回显
   ├─ 显示进度：✅ 已提交 → ⏳ 审核中 → ⏳ 待完成
   ├─ 预计 3-7 个工作日
   └─ 李四可以随时回来这个页面查看进度

5️⃣ 🔔 审核通过（通常在 3-7 天内）
   ├─ 李四收到系统通知
   ├─ 回到 Profile → Settings
   ├─ 发现现在显示 【切换身份】选项
   └─ 点击切换 → 变成管理员

6️⃣ 切换后，导航栏变为：Explore | **Manage** | Messages | Profile

7️⃣ 进入 s-manage (管理台首页) - 李四的工作中心
   ├─ 顶部显示：🏛️ 校园戏剧社管理员
   ├─ 中间显示 4 个数据卡片：
   │  ├─ 主页访问：1,284 （↑23% 本周）
   │  ├─ 收藏量：342
   │  ├─ 被匹配次数：567 （用户通过智能匹配看到我们）
   │  └─ 报名人数：28（待审核 20）
   └─ 下面是 4 个快捷操作卡片

8️⃣ 李四想发布招新公告
   ├─ 点击快捷卡 【📢 发布招新】
   ├─ 进入 s-publish (发布招新页)
   │  ├─ 开始写公告
   │  │  ├─ 标题：2024年秋季招新
   │  │  ├─ 封面：上传海报
   │  │  ├─ 招新部门：演出部、编剧部、舞美部、宣传部（可多选）
   │  │  ├─ 时间安排：报名期限、面试时间、录取时间
   │  │  ├─ 招募人数：30人
   │  │  ├─ 要求说明：零基础欢迎、热情最重要
   │  │  └─ 联系方式
   │  ├─ 点击 【预览】查看效果
   │  └─ 点击 【发布】
   └─ 公告自动推送到 Explore 首页，让所有用户能看到

9️⃣ 李四想查看和审核报名
   ├─ 点击快捷卡 【📋 查看报名】
   ├─ 进入 s-applications (报名表管理页)
   │  ├─ 看到一个表格，列出所有报名人
   │  ├─ 搜索框：可以搜索某个学生的名字
   │  ├─ 筛选选项：全部 | 待处理 | 已通过 | 未录取
   │  ├─ 导出按钮：可以下载 Excel
   │  └─ 每行是一个报名人：
   │     ├─ 头像、姓名、院系、联系方式
   │     ├─ 报名部门
   │     └─ 【查看详情】【标记通过】【标记未录取】
   └─ 李四逐一审核后标记状态，系统自动通知用户

🔟 李四想回复用户的疑问
   ├─ 点击快捷卡 【💬 管理留言】
   ├─ 进入 s-faq (FAQ/留言管理)
   │  ├─ 看到两部分：
   │  │  ├─ 「预设 FAQ」：社团设置的常见问题
   │  │  └─ 「用户留言」：用户通过社团主页 s-club 提的问题
   │  ├─ 筛选选项：全部 | 待回复 | 已回复
   │  ├─ 每条留言显示：
   │  │  ├─ 提问人名字
   │  │  ├─ 问题内容

   │  │  └─ 【查看详情】【回复】

1️⃣1️⃣ 李四还可以编辑社团主页

1️⃣2️⃣ 李四可以查看数据统计

   ├─ 进入 s-stats (数据统计页)
   │  ├─ 显示几个核心指标卡片：

1️⃣3️⃣ 李四还可以查看 Messages
```

## 🎯 快速决策树

### Q：我应该如何进入系统？

```
你想...

1️⃣ 先随便逛逛，不登录
   → 点击"先逛逛" → 进入 Explore 只读模式（游客模式）

2️⃣ 我是学生，想找社团
   → 点击"我是新生" → 登录/注册 → 进入完整功能（新生模式）

3️⃣ 我是社团负责人
   → 点击"我是社团负责人" → 登录/注册 → 填写认证 → 等待通过 → 进入管理模式
```

### Q：什么操作会被限制？

```
游客状态下：
❌ 【收藏】→ 弹窗"请先登录"
❌ 【报名】→ 弹窗"请先登录"
❌ 【提问】→ 弹窗"请先登录"
❌ 【做匹配】→ 弹窗"请先登录"
❌ 【进入 Match】→ 无此导航栏
❌ 【进入 Profile】→ 无此选项
❌ 【Messages】→ 无此功能

新生状态下：
❌ 【编辑社团主页】→ 只有你管理的社团才显示✏️按钮
❌ 【发布公告】→ 无此权限
❌ 【查看报名表】→ 无此权限
❌ 【查看数据統計】→ 无此权限
```

### Q：我想从新生升级成管理员，怎么做？

```
1️⃣ Profile → 设置 → 【申请社团管理员】

2️⃣ 填写认证表单
   ├─ 社团名称
   ├─ 职位
   ├─ 身份证件
   ├─ 证书文件
   └─ 联系电话

3️⃣ 【提交申请】

4️⃣ 等待 3-7 个工作日审核

5️⃣ 🔔 审核通过后
   ├─ 收到通知
   ├─ Profile → Settings 显示【切换身份】
   └─ 点击切换 → 变成管理员 → 解锁 Manage 和所有管理功能
```
